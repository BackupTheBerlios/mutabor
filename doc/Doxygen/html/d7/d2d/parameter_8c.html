Generiert eine Interpreter-Parameterliste aus eine Argumentliste.

Dabei wird eine Bezugsliste verwendet, die für parametrische argumente die Wertefelder zur Verfügung stellt. Taucht ein unerwartetes Ereignis auf, wird ein fataler Fehler ausgelöst.

Parameter
listeListe der Argumente, die in die Interpreter-Liste eingetragen werden sollen.
bezugs_listeListe mit den Bezügen/Wertefeldern, die für nichtnumerische Argumente verwendet werden soll.
