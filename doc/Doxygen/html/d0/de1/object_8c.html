int mutabor_check_class ( void *  _self,
void *  object 
)

Checks, if a pointer is a subclass of a given class.

Parameter
_selfobject to be checked
objectpointer to the virtual table of the desired class.
Rückgabewerte
0if not an object of the desired class
1if _self is an instance of object
Rückgabe
The function does not return if it encounters cyclic dependencies of objects (or what it takes as objects). In most cases of weired data structures it will (hopefully) fail with a null pointer dereference or a segmentation fault.
